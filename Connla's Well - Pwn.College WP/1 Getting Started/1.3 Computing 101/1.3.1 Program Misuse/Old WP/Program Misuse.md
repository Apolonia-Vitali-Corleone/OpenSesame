# SUID

一种特殊权限。



# linux command:id

```bash
gao@fossa:~$ id
uid=1000(gao) gid=1000(gao) groups=1000(gao),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),120(lpadmin),132(lxd),133(sambashare)
```

`uid`是user id，

`gid`是当前user的工作group

`groups`是包含该user的所有组。

上图可以看出：

- 用户是gao

- 当前工作组是gao

- 用户gao所属的组是

  gao,adm,cdrom,sudo,dip,plugdev,lpadmin,lxd,sambashare



# linux command:grep

## -v

参数：`-v`

作用：排除匹配的内容

```bash
-v, --invert-match
Invert the sense of matching, to select non-matching lines.
```
